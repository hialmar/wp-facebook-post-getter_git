Easily Import Multiple Facebook Posts in your Wordpress

== Description ==

Based on WP Create Multiple Posts & Pages by Sajjad Hossain Sagor (sajjad67)

This plugin allows to import multiple posts from Facebook at once.

= Features: =
- Download Facebook Posts
- Add multiple posts
- Add multiple pages
- Set post status
- Set post author
- Set post category (for posts only)
- Custom Post Types (CPT) Supported

== Installation ==
To add a WordPress Plugin using the built-in plugin installer:

Go to Plugins > Add New.

1. Type in the name "WP Facebook Post Getter" in Search Plugins box
2. Find the "WP Create Multiple Posts & Pages" Plugin to install.
3. Click Install Now to begin the plugin installation.
4. The resulting installation screen will list the installation as successful or note any problems during the install.
If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.

To add a WordPress Plugin from github repo / plugin zip file :
1. Go to wordpress plugin page
2. Click Add New & Upload Plugin
3. Drag / Click upload the plugin zip file
4. The resulting installation screen will list the installation as successful or note any problems during the install.
If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.

== Frequently Asked Questions ==
= How to use this plugin? =
Just after installing the plugin, Go to settings and you'll see the option in the menu.

Choose the from and to dates and the number of posts and click on Add Posts.

Note that you can select the post categories, pages instead of posts...
